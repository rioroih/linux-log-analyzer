# Linux Log Analyzer

A Bash-based CLI tool that analyzes Linux system logs and generates reports.

## Features
- Count system errors
- Count warnings
- Display recent log errors
- Generate automated log reports
- Menu-driven CLI interface

## Technologies Used
- Bash Scripting
- Linux (CentOS Stream 9)
- Git & GitHub

## Usage

Make script executable:

chmod +x log_analyzer.sh

Run the tool:

./log_analyzer.sh

## Example Output
The tool analyzes logs from:
/var/log/messages

It generates:
log_report.txt
